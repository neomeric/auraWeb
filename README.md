# AuraFusen Website

A futuristic, high-tech website for AuraFusen - an advanced AI technology that fuses emotion with intelligence.

## Overview

This website is designed with a mysterious, futuristic aesthetic to showcase AuraFusen as a cutting-edge AI technology breakthrough rather than a commercial product. The design follows a minimalist approach with a dark theme, subtle animations, and interactive elements.

## Features

- Responsive design that works on all devices
- Dark mode with futuristic UI elements
- Interactive animations showcasing the technology
- Waitlist signup functionality
- Smooth scroll animations and transitions

## Structure

The website consists of the following sections:

1. **Hero Section** - First impression with bold statement about AuraFusen
2. **What is AuraFusen?** - Technology description with animated visualization
3. **Use Cases** - Interactive cards showing potential applications
4. **How It Works** - Technical overview of the AuraFusen architecture
5. **Sign-Off** - Final call-to-action with waitlist signup

## Technologies Used

- HTML5
- CSS3 (with modern features like CSS variables, flexbox, grid)
- Vanilla JavaScript (no frameworks)
- SVG animations

## Running the Website

To view the website locally, simply open the `index.html` file in a web browser or run a local server.

```bash
# Example using Python's built-in HTTP server
python -m http.server
```

Then navigate to `http://localhost:8000` in your browser.

## Design Choices

- **Color Scheme**: Purple, blue, and pink gradient representing the fusion of different AI capabilities
- **Typography**: Clean, modern sans-serif font (Inter) for optimal readability
- **Animations**: Subtle animations that enhance the futuristic feel without being distracting
- **Dark Theme**: Creates a mysterious, high-tech atmosphere aligned with cutting-edge AI
